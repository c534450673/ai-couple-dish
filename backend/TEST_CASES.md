# 情侣私密菜单系统 - 测试用例文档

## 测试概述

本文档描述了情侣私密菜单（AI Couple Dish）系统的完整测试用例，覆盖单元测试、集成测试、安全测试和API测试。

## 测试环境

- **Java版本**: 17
- **Spring Boot版本**: 2.7.18
- **数据库**: MySQL 8.0
- **缓存**: Redis 7
- **测试框架**: JUnit 5 + Mockito + Spring Boot Test

## 测试文件结构

```
backend/src/test/java/com/aicoupledish/
├── UserServiceTest.java          # 用户服务测试 (11个测试用例)
├── MenuServiceTest.java         # 菜单服务测试 (22个测试用例)
├── AnniversaryServiceTest.java  # 纪念日服务测试 (19个测试用例)
├── FeedServiceTest.java         # 投喂服务测试 (26个测试用例)
├── NoteServiceTest.java         # 笔记服务测试 (16个测试用例)
├── CoupleServiceIntegrationTest.java  # 情侣服务集成测试 (18个测试用例)
├── SecurityTest.java            # 安全测试 (25个测试用例)
└── ApiIntegrationTest.java       # API集成测试 (35个测试用例)
```

**总计**: 172+ 测试用例
