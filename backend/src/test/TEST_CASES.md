## 1. 测试概述

### 1.1 测试范围

| 模块 | 优先级 | 测试类型 |
|------|--------|----------|
| 情侣关系模块 | P0 | 功能测试、边界测试 |
| 私密菜单模块 | P0 | 功能测试、业务规则测试 |
| 投喂模块 | P1 | 功能测试、状态流转测试 |
| 纪念日模块 | P1 | 功能测试、计算逻辑测试 |

### 1.2 测试环境要求

- JDK 11+
- MySQL 8.0+
- Redis 6.0+
- Spring Boot 2.7.x
- JUnit 5 + Mockito

### 1.3 测试数据说明

- 测试数据位于 `src/test/resources/test-data.sql`
- 所有数据为后端数据库初始语句
- 不使用前端mock数据
